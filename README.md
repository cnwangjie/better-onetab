
# Extension Manager (Chrome)

一键管理Chrome浏览器的扩展，操作简单方便，减少扩展对资源的占用及浪费，绿色安全。

* 操作简单快捷
* 安全无风险
* 提升浏览性能


## Feature

* 分组管理
* 扩展锁定
* 闪电操作
* 智能排序
